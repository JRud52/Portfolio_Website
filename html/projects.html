Project Markdown

The point of this project is to create a markdown editor for students. As a computer science student myself, I wanted an application that I could use to create class-notes and study-notes that was simple and quick to use. I looked for a program that had the following necessary features:

  • Cross platform
  • Offline storage with optional online backups to google drive/dropbox
  • Responsive interface that could be reconfigured based on the user's preference
  • Ability to switch between the editor and trhe preview easily
  • Ability to split the view to see both the preview and the editor
  • Supported extra markdown features like table of contents, tables, etc.
  • Custom css rules for the preview/output
  • export to various formats
  • ability to open a folder of markdown files and switch between them easily
  • tabbed editor to edit multiple files at once
So far I have not been able to find an application that does everything that I want, so I decided to write my own that was geared towards students. For the student, I would like to add the following features:
  • The user may create a "Study Note" that adds the contents of each other file in the folder to it in order. Eventually the program may be able to automatically generate this file as the user creates more markdown files.
  • Any heading marked Lecture 1 or Chapter 1 will be added to the file in order.
  • The will be able to make special headings like # Definitions and have the program look for matching headings in other files. If it finds another # Definitions heading, then it will automatically add all the contents of that section to the study note.
